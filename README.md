## Business Sales Performance Analytics Dashboard

## Overview

This project analyzes business sales performance using Power BI and the Sample Superstore dataset.

## Dashboard Metrics

* Total Sales: 2.30M
* Total Profit: 286.40K
* Profit Margin: 12.47%
* Total Orders: 5K

## Visualizations

* Sales Trend by Order Date
* Sales by Category
* Sales by Region
* Top 10 Products by Sales
* Top 10 Customers by Sales

## Tools Used

* Power BI Desktop
* Microsoft Excel
* GitHub

## Key Insights

* Technology category generated the highest sales.
* West region recorded the highest sales.
* Total profit reached 286.40K.
* Profit margin was 12.47%.
